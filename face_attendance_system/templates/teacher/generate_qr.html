QR Code Settings
How to Use
  1. Select the subject and section
  2. Set QR expiry time (for this class)
  3. Click "Generate QR Code"
  4. A new random QR code is created for each class
  5. Display the QR code on your screen during class
  6. Students scan with their mobile devices
  7. Students must be within 200m of KEC campus
Student Instructions

Students should:

  • Login to the attendance system
  • Go to "Mark Attendance" page
  • Allow camera and location access
  • Scan the QR code displayed by the teacher
  • If "You're out of location" appears, move closer to campus
  • Attendance only marks if within 200m of KEC
